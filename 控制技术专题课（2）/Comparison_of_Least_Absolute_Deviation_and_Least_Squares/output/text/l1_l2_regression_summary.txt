最小一乘回归与最小二乘回归对比分析
====================================

任务覆盖内容:
1. 线性参数拟合案例
2. 三次多项式系数拟合案例
3. 高斯噪声场景与异常点场景的鲁棒性分析
4. 图像导出与结果说明

线性拟合 - 高斯噪声
-----------
最小二乘: 参数误差 = 0.103, 训练RMSE = 0.804, 训练MAE = 0.663, 真实曲线RMSE = 0.164
最小一乘: 参数误差 = 0.181, 训练RMSE = 0.820, 训练MAE = 0.638, 真实曲线RMSE = 0.318

线性拟合 - 含异常点
-----------
最小二乘: 参数误差 = 0.139, 训练RMSE = 2.420, 训练MAE = 1.284, 真实曲线RMSE = 0.157
最小一乘: 参数误差 = 0.158, 训练RMSE = 2.444, 训练MAE = 1.246, 真实曲线RMSE = 0.282

多项式拟合 - 高斯噪声
------------
最小二乘: 参数误差 = 0.174, 训练RMSE = 0.824, 训练MAE = 0.606, 真实曲线RMSE = 0.126
最小一乘: 参数误差 = 0.250, 训练RMSE = 0.857, 训练MAE = 0.594, 真实曲线RMSE = 0.188

多项式拟合 - 含异常点
------------
最小二乘: 参数误差 = 0.088, 训练RMSE = 2.715, 训练MAE = 1.414, 真实曲线RMSE = 0.435
最小一乘: 参数误差 = 0.250, 训练RMSE = 2.758, 训练MAE = 1.279, 真实曲线RMSE = 0.188

Monte Carlo 鲁棒性结论
----------------------
线性模型噪声扫描: 最小二乘平均RMSE = 0.230, 最小一乘平均RMSE = 0.279
线性模型异常点扫描: 最小二乘平均RMSE = 0.581, 最小一乘平均RMSE = 0.164
多项式模型噪声扫描: 最小二乘平均RMSE = 0.350, 最小一乘平均RMSE = 0.436
多项式模型异常点扫描: 最小二乘平均RMSE = 0.945, 最小一乘平均RMSE = 0.393

简要结论:
1. 在仅含高斯噪声时，最小二乘通常具有更优或相近的均方误差表现。
2. 在存在显著异常点时，最小一乘的拟合曲线偏移更小，测试RMSE增长更慢，鲁棒性更强。
3. 对高阶多项式拟合而言，异常点会放大最小二乘的参数偏移，而最小一乘对局部极端样本更不敏感。
